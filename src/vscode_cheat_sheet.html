My Typical Initial Setup

Initial Setup for NPM and .NET Core Support.
  1. Install latest version of .NET core (https://www.microsoft.com/net/download).
  2. Install latest stable version of Node.js (https://nodejs.org).
  3. Install VS Code extension for NPM support (eg2.vscode-npm-script)
    This can be done by searching for the extension or by entering the following into the VS Code terminal:
    code --install-extension eg2.vscode-npm-script
Initializing web app project
mkdir [Project Name]
cd [Project Name]
dotnet new web --framework netcoreapp2.0
dotnet sln ..\[Solution Name].sln add .\[Project Name].csproj
npm init
npm install jquery
npm install popper.js
npm install bootstrap
npm install bootstrap-table
npm install bootstrap-datetime-picker
After packages are installed, the JavaScript and CSS sources can be copied out of the node_modules folder, and into the wwwroot/lib folder (creating it, if necessary).
